# Walmart-Sales-Performance-Analysis


##  Project Title  
**Walmart Sales Performance Strategy – Data-Driven Insights for Smarter Store Investment**

---

##  Problem Statement  
Retail chains need to identify:
- High-performing stores  
- Impact of economic conditions on sales  
- Stores with stable and predictable demand  

This project provides **data-driven insights** to support smarter investment and expansion decisions.

---

##  Dataset Overview  
- **Time Period:** 2010 – 2012  
- **Total Stores:** 45  
- **Records:** ~6400  
- **Fields:**
  - Store  
  - Date  
  - Weekly Sales  
  - Holiday Flag  
  - Temperature  
  - Fuel Price  
  - CPI  
  - Unemployment  

---

##  Tech Stack  
- SQL (MySQL)  
- Power BI  
- Data Analytics
